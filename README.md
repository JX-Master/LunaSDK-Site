# Luna SDK Site

This repository hosts source files for [Luna SDK Official Site](https://www.lunasdk.net).

## Building
1. Install python.
1. Install mkdocs:
    ```shell
    pip install mkdocs
    ```
1. Build site.
    ```shell
    mkdocs build
    ```
1. Deploy to GitHub pages
    ```shell
    mkdocs gh-deploy
    ```
